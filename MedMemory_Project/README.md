# MedMemory

AI-powered personal health memory platform.

## Features
- Upload medical reports
- OCR extraction
- AI-powered health timeline
- AI doctor summary
- AI chat from personal records
- Family health management

## Tech Stack
- Frontend: Next.js + Tailwind
- Backend: FastAPI
- Database: PostgreSQL
- AI Layer: OpenAI + LangChain
- OCR: Tesseract
